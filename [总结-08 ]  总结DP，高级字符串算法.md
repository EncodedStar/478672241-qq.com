## 复习动态规矩

**DP 顺推模板**
```python
function DP():
     dp = [][] # 二维情况
     for i = 0 ... M {
          for j = 0 ... N {
               dp[i][j] = _Function(dp[i'][j'] ...)
          }
     }
     return dp[M][N]
```

**关键点**
- 动态规划 和 递归 或者 分治 没有根本上的区别（关键看有无最优的子结构）
- 拥有共性：找到重复子问题
  
**差异性**
- 最优子结构、中途可以淘汰次优解

## 高级DP

**高级DP为什么复杂？**
1. 状态拥有更多维度（二维、三维、或者更多、甚至需要压缩）
2. 状态方程更加复杂
